# MSI - GS63 - 7代 笔记本 EFI项目
hackintosh: gs63-7re the efi files of devices....

黑苹果 MSI G*62/63 - 7* 笔记本的 EFI维护项目(*代表任意字符)
	因MSI的该系列主板采用相同的配置的内建硬件,故可以通用.*详细的的通用将在支持机型里面说明

支持的操作系统级别:
macOS 10.13.6 (推荐使用)
	使用该系统支持 本机自带的独立显卡,支持GPU显卡加速
macOS 10.14
	支持独立显卡,但是不支持GPU显卡加速
macOS 10.15
	不支持独立显卡,使用核心显卡
